# Installation for Kali Linux

  ```
  apt update && apt upgrade
  apt install git python
  pip install bs4 requests 
  git clone https://github.com/afrizalwahyuadi66/UAS-Pemlan.git
  ```

# Use
  
  ```
  cd UAS-Pemlan
  python main.py
  ```
